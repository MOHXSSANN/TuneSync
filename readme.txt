

INSTALL INSTRUCTIONS: 

Navigate to the server.js directory
In the terminal type "npm install"

LAUNCH INSTRUCTIONS: Please Open your terminal and navigate to the server directory, 
then type "node server.js" to start running the server. Please then click on the link or paste any of these link in your browser:
http://localhost:3000/mytunes.html
http://localhost:3000/mytunes
http://localhost:3000/index.html
http://localhost:3000/
http://localhost:3000

TESTING INSTRUCTIONS: 
after you have visitied any of the following links you can test the program by entering a song name
and adding it to the playlist by using the "+" button and using the up and down buttons to move the 
songs queue up and down the playtlist. you can also remove a song from your playlist by hitting the - button
